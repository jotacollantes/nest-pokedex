# Ejecutar en Desarrollo

1. Clonar el repo
2. Ejecutar:
```
yarn
```
3. Tener Nest CLI instalado
```
yarn global add @nestjs/cli
```
4. Levantar la base de datos (archivo de configuracion ./docker-compose.yaml)
```
docker-compose up -d
```
